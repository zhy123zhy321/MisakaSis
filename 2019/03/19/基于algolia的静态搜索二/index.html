# 基于Algolia的静态搜索（二）
  1. 先安装node.js Node官网

各系统安装不同就不在这里说了

  1. 创建一个目录

  2. 执行npx

   npx create-instantsearch-app ais-ecommerce-demo-app
  1. 选择InstantSearch.js 回车确认
   npx: installed 146 in 20.221s
   
   Creating a new InstantSearch app in ais-ecommerce-demo-app.
   
   ? InstantSearch template (Use arrow keys)
     Web
   > Angular InstantSearch
     InstantSearch.js
     React InstantSearch
     Vue InstantSearch
     Mobile
     InstantSearch Android
     InstantSearch iOS
     React InstantSearch Native
  1. 选择版本
   ? InstantSearch template InstantSearch.js
   ? InstantSearch.js version (Use arrow keys)
     Latest stable version (recommended)
   > 3.2.1
     All versions
     3.2.1
     3.2.0
     3.1.1
     3.1.0
   (Move up and down to reveal more choices)
  1. 接下来输入你的Application ID等
   ? InstantSearch template InstantSearch.js
   ? InstantSearch.js version 3.2.1
   ? Application ID 7MXMJHTAW7
   ? Search API key 1d9ed3a31ec6ef9514f2ad7d174a97c9
   ? Index name MisakaSis
  1. 选择需要显示的字段，上下翻页，空格选择，回车确认
     Used to generate the default result template
    ──────────────
    From your index
    (*) title
    (*) description
    ( ) uri
   >( ) content
    (*) date
    (*) banner
    (*) categories
    ( ) images
   (Move up and down to reveal more choices)
  1. 接下来会提示两个警告，大概就是版本被弃用了，有更改好的替代，不用管。
   npm WARN deprecated circular-json@0.3.3: CircularJSON is in maintenance only, fl
   atted is its successor.
   npm WARN deprecated browserslist@1.7.7: Browserslist 2 could fail on reading Bro
   wserslist >3.0 config used in other tools.
  1. 成功提示
      Created ais-ecommerce-demo-app at ais-ecommerce-demo-app.
   
   Begin by typing:
   
     cd ais-ecommerce-demo-app
     npm start
   
   ⚡️  Start building something awesome!
  1. 按上面说的打开目录，运行npm。
